# Freaky Fashion

Backend-projekt utvecklat med Node.js, Express, EJS och SQLite.

## Funktioner

- Visa produkter
- Visa produktdetaljer
- Visa kategorier
- Sök produkter
- Adminpanel för produkter
- Adminpanel för kategorier
- Skapa nya produkter
- Skapa nya kategorier
- Bilduppladdning med Multer

## Tekniker

- Node.js
- Express.js
- EJS
- SQLite
- Multer
- HTML
- CSS
- JavaScript

## Installation

Installera beroenden:

```bash
npm install
```

Starta applikationen:

```bash
npm run dev
```

Öppna sedan:

```text
http://localhost:3000
```

## Databas

Databasen använder SQLite.

Återskapa databasen:

```bash
node db/run-sql.js
```
